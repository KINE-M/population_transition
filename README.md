# アプリ名

都道府県別　総人口推移グラフ

# 内容

- 選択した都道府県の人口推移が折れ線グラフで表示される

# 技術スタック

- フロントエンド
  - React
  - Typescript
  - Recharts（グラフ化するライブラリ）
  - axios
- その他
  - Docker
  - Prettier
  - Eslint
  - sass
  - api → RESAS 地域経済分析システムの api を利用しています。
